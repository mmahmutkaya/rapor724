**Gerekli akış:**
1. Proje sahibi `/ihale` sayfasından "Firma Davet Et" butonuna basar
2. E-posta + iş paketi seçer
3. Sistem benzersiz token üretir → `ihale_invitations` tablosuna kaydeder
4. Firma temsilcisi linke tıklar → kayıt/giriş → `work_package_members`'a otomatik eklenir
5. Firma `/ihale/pozlar`'da kendi birim fiyat sütununu görür ve doldurur

**Gerekli Supabase tablosu:**
```sql
CREATE TABLE ihale_invitations (
  id          uuid PRIMARY KEY DEFAULT gen_random_uuid(),
  token       text UNIQUE NOT NULL DEFAULT gen_random_uuid()::text,
  work_package_id uuid NOT NULL,
  email       text NOT NULL,
  invited_by  uuid NOT NULL REFERENCES auth.users(id),
  accepted_at timestamptz,
  created_at  timestamptz DEFAULT now()
);
```

### Supabase Tabloları — HENÜZ OLUŞTURULMADI

```sql
-- 1. Proje üyelik + rol tablosu
CREATE TABLE project_members (
  id              uuid PRIMARY KEY DEFAULT gen_random_uuid(),
  project_id      uuid NOT NULL REFERENCES projects(id) ON DELETE CASCADE,
  user_id         uuid REFERENCES auth.users(id) ON DELETE SET NULL,
  email           text NOT NULL,
  role            text NOT NULL CHECK (role IN ('admin', 'metraj_hazirlayan', 'metraj_onaylayan')),
  work_package_id uuid REFERENCES work_packages(id) ON DELETE CASCADE,
  invited_by      uuid REFERENCES auth.users(id),
  created_at      timestamptz DEFAULT now(),
  UNIQUE NULLS NOT DISTINCT (project_id, email, role, work_package_id)
);

-- 2. İhale davetleri
CREATE TABLE ihale_invitations (
  id              uuid PRIMARY KEY DEFAULT gen_random_uuid(),
  token           text UNIQUE NOT NULL DEFAULT gen_random_uuid()::text,
  work_package_id uuid NOT NULL REFERENCES work_packages(id) ON DELETE CASCADE,
  email           text NOT NULL,
  invited_by      uuid NOT NULL REFERENCES auth.users(id),
  accepted_at     timestamptz,
  accepted_by     uuid REFERENCES auth.users(id),
  created_at      timestamptz DEFAULT now()
);
```

### Sayfanın 3 Sekmesi
- `/ekip` (default) → **Proje Yetkilileri**: admin + proje geneli hazırlayan/onaylayan
- `/ekip?s=is-paketleri` → **İş Paketi Rolleri**: work package seç + role ata
- `/ekip?s=ihale` → **İhale Davetleri**: email + iş paketi → token üret + link kopyala
