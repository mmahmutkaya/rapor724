## Mevcut Durum

Sayfa **tamamen çalışır durumda**. Aşağıdaki özellikler implemente edildi:

### Ağaç Yapısı (Hiyerarşik Node)
- `formNodes`: `[{ id, parent_id, name, isFixed, order_index, rows: [{id, name, isWp, butce}] }]`
- `flattenTree()` + `buildDisplayTree` yardımcıları ile depth-first sıralama
- `NODE_PALETTE` — pozlar/index.js ile birebir aynı (8 renk, depth % 8)
- Leaf node = çocuğu olmayan node → yeşil nokta (●) gösterir
- Non-leaf node = collapse/expand toggle ile ▶/▼

### Node İşlemleri (AppBar butonları)
- **Yukarı/Aşağı**: `order_index` swap ile sibling sıralama
- **Sol'a**: parent seviyesine çıkar (grandparent'ın çocuğu olur)
- **Sağ'a**: bir üst kardeşin son çocuğu olur
- **Alt Düğüm Ekle**: aktif node'un çocuğu olarak ekler
- **Düğüm Ekle (kök)**: root seviyesine yeni node
- **Sil**: node + tüm torunlarını siler

Buton aktiflik koşulları:
```
canMoveUp    = !isFixed && sibIdx > 0
canMoveDown  = !isFixed && sibIdx < siblings.length - 1
canMoveLeft  = !isFixed && parent_id != null
canMoveRight = !isFixed && sibIdx > 0
```
"Düğüm Ekle" butonu: aktif node'un rows'u varsa veya leaf değilse disabled.

### Fixed Node — İş Paketleri
- `id: "node_ispaketleri"`, `isFixed: true`, `parent_id: null`
- `rows[]` = `workPackages` (Supabase'den `useGetWorkPackages` hook'uyla çekilir)
- `isWp: true` olan row'lar TextField göstermez, Typography gösterir
- Taslak yüklenince iş paketi listesi güncel tutulur (`buildWpRows` merge logic)

### Satır İşlemleri (leaf node seçiliyken)
- `addRow(nodeId)`: leaf node'a yeni boş satır ekle
- `updateRow(nodeId, rowId, field, val)`: ad veya butce güncelle
- `deleteRow(nodeId, rowId)`: isWp olmayan satırı sil

### Bütçe Hesaplama
- `nodeRowTotal(node)` = node.rows toplamı
- `subtreeTotal(nodeId)` = recursive (kendi rows + tüm torunlar)
- `grandTotal` = tüm root node'ların subtreeTotal toplamı
- `fmt(v)` = `tr-TR` locale, 2 hane (Intl.NumberFormat)

### Auto-Save (localStorage Taslak)
- `draftKey = "butce_draft_{selectedProje.id}"`
- `useEffect`: her `formNodes`/`formAciklama` değişiminde 600ms debounce ile localStorage'a yazar
- `draftInfo: { savedAt: ISO string }` — bilgi olarak state'de tutulur
- `openForm()`: localStorage'dan taslak varsa yükler; iş paketi row'larını merge eder
- `cancelForm()`: localStorage'daki taslağı **silmez** (kaldığı yerden devam edilebilsin)
- `discardDraft()`: taslağı localStorage'dan siler ve draftInfo'yu temizler

### Versiyon Kaydetme
Supabase `budget_versions` tablosu:
```js
{
  project_id, versiyon_number, aciklama,
  total_kesif_tutar: null,  // henüz kullanılmıyor
  total_butce_tutar: grandTotal,
  is_paketler_satirlar: {
    nodes: [{ nodeId, nodeName, isFixed, parentId, orderIndex, rows: [...], totalButce }]
  },
  created_by: appUser.id
}
```
Kayıt sonrası taslak silinir, form kapanır, versiyon listesi yenilenir.
