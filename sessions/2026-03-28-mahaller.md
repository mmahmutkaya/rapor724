## Görünüm Modları (viewMode)

Üç mod arası döngüsel geçiş:

| Mod | Etiket | Açıklama |
|-----|--------|----------|
| `lbsOnly` | LBS | Sadece LBS ağacı, mahaller gösterilmez |
| `lbsMahal` | L+M | LBS ağacı + altında mahal satırları — **varsayılan** |
| `mahalOnly` | Mahal | Düz mahal listesi, LBS chip filtreleriyle |

**Kısıt**: `rawLbsNodes.length === 0` → toggle butonu **disabled** (L+M ve Mahal moduna geçilemez).
LBS boşken mod otomatik `lbsOnly`'e sıfırlanır (useEffect).

flattenTree(nodes, parentId, depth)   // depth-first, order_index sıralı
nodeColor(depth)                      // 8 renk paleti (depth % 8) — FARKLI palet (pozlar'dan farklı tonlar)

### isLeafSet
LBS'de çocuğu olmayan node'lar yaprak kabul edilir.

### Collapse/Expand
- `collapsedIds: Set<id>` — toggle ile açılıp kapanır
- `isHiddenByAncestor(node)` — parent zincirinde collapsed varsa gizler

### İnteraksiyon (lbsOnly ve lbsMahal)
- **İsim alanı tıklama** → collapse/expand (tüm node'lar)
- **Sağ alan tıklama** → seç/iptal (`activeLbsNodeId` toggle), sarı nokta görünür
- **Satır rengi seçimde değişmez** — her zaman `c.bg`

### Yaprak Node Renk Göstergesi
| Renk | Anlam |
|---|---|
| Yeşil nokta `#65FF00` | Mahal içermiyor — alt başlık eklenebilir |
| Kırmızı nokta `#FF4444` | Mahal içeriyor — alt başlık **eklenemez** |

## LBS Node İşlemleri

### Oluşturma (`+` butonu)

| Durum | Aksiyon |
|-------|---------|
| `mahalOnly` modu | Disabled |
| `lbsOnly` + seçim yok | `show = 'LbsChildCreate'` → kök başlık formu |
| `lbsOnly` + node seçili + **mahal içeriyor** | ❌ Uyarı dialog, form açılmaz |
| `lbsOnly` + node seçili + mahal içermiyor | `show = 'LbsChildCreate'` → alt başlık formu |
| `lbsMahal` + node seçili değil | Disabled |
| `lbsMahal` + node seçili | `show = 'MahalCreate'` |

**Çift güvence**: `+` onClick + `handleAddLbsChild()` her ikisinde de "mahal içeriyor" kontrolü var.

`LbsChildCreate` formu: Kod (isteğe bağlı) + Başlık Adı alanları. Başlık: seçim varsa "Alt Başlık Ekle — {node adı}", yoksa "LBS Kök Başlık Ekle".

Supabase: `lbs_nodes.insert({ project_id, parent_id, name, code_name, order_index })`

## Mahal İşlemleri

### Oluşturma
- `lbsMahal` modunda yaprak LBS node seçiliyken `+` → `MahalCreate` formu

### Silme (satır başındaki delete ikonu)
- `handleDeleteMahal(mahal)` — DialogAlert onay sonrası `supabase.from('work_areas').delete().eq('id', mahal.id)`

### Düzenleme
- Satırdaki Edit ikonu → `editingMahal` state set → `FormMahalEdit` gösterilir
