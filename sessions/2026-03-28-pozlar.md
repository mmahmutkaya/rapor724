### isLeafSet
```js
// Child'ı olmayan node'lar yaprak
const isLeafSet = useMemo(() => {
  rawWbsNodes.forEach(n => {
    if (!rawWbsNodes.some(c => c.parent_id === n.id)) s.add(n.id)
  })
}, [rawWbsNodes])
```

### Collapse/Expand
- `collapsedIds: Set<id>` — toggle ile açılıp kapanır
- `isHiddenByAncestor(node)` — parent zincirinde herhangi bir collapsed varsa gizler
- **wbsPoz modunda**: tüm node'lar (leaf dahil) isim alanına tıklayınca collapse/expand yapar (▼/▶)
- **wbsOnly modunda**: sadece non-leaf node'lar collapse/expand yapar; leaf'ler renkli nokta gösterir

## WBS Node İşlemleri (AppBar — activeWbsNodeId seçiliyken)

```
canMoveUp    = siblingIdx > 0
canMoveDown  = siblingIdx < siblings.length - 1
canMoveLeft  = parent_id != null
canMoveRight = siblingIdx > 0
```

- **Yukarı/Aşağı**: kardeşlerle `order_index` swap
- **Sol'a**: grandparent'ın çocuğu olur, parent'ın altındaki kardeşlerin order_index'i kaydırılır
- **Sağ'a**: bir üst kardeşin son çocuğu olur

## Poz/WBS Ekleme Akışı

AppBar `+` butonu davranışı:

| Durum | Aksiyon |
|-------|---------|
| `wbsOnly` + seçim yok | `show = 'WbsChildCreate'` → kök başlık formu |
| `wbsOnly` + node seçili + **poz içeriyor** | ❌ Uyarı dialog, form açılmaz |
| `wbsOnly` + node seçili + poz içermiyor | `show = 'WbsChildCreate'` → alt başlık formu |
| `wbsPoz` + non-leaf seçili + **poz içeriyor** | ❌ Uyarı dialog, form açılmaz |
| `wbsPoz` + non-leaf seçili + poz içermiyor | `show = 'WbsChildCreate'` → alt başlık formu |
| `wbsPoz` + node seçili değil | Disabled |
| `wbsPoz` + leaf seçili + birim yok | Disabled (Proje Ayarları uyarısı) |
| Diğer durumlar | `show = 'PozCreate'` |

**Çift güvence**: `+` onClick + `handleAddWbsChild()` her ikisinde de "poz içeriyor" kontrolü var.

## Seçim Modu (selectMode)

- `selectMode: bool` + `selectedIds: Set<pozId>`
- Aktifken: AppBar'da ✕ (iptal) + ✏️ (1 seçiliyse düzenle) + 🗑 (sil) butonları

- `pozOnly` modunda: header checkbox ile tüm filtrelenmiş pozları seç/kaldır
